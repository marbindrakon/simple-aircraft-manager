Microsoft Edge WebView2 Runtime (Windows only)
================================================================

Simple Aircraft Manager uses the Microsoft Edge WebView2 Runtime to display
its user interface on Windows. The WebView2 Runtime is not bundled with this
product; it is a separate Microsoft component installed by the operating
system or downloaded by the user. Download and license terms are available
from Microsoft at:

  https://developer.microsoft.com/en-us/microsoft-edge/webview2/

================================================================
PyInstaller bootloader
================================================================

The desktop bundle is built with PyInstaller, which embeds a small native
"bootloader" binary alongside the application. The bootloader is licensed
under the GPLv2 with an explicit exception that permits redistribution as
part of non-GPL applications. The bootloader is the only component of
PyInstaller that ships in the bundle; PyInstaller itself is a build-time
dependency. See:

  https://github.com/pyinstaller/pyinstaller/blob/develop/COPYING.txt
